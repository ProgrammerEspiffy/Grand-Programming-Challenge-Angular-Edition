English

To run the project as a test:

1. Open Terminal: It can be CMD or Git Bash

2. Put This Command: "cd Project-Programmermoid"

3. Run this "ng serve" Command

4. Open the Browser and put this Link:"http://localhost:4200/"

5. Enjoy The Project :)
